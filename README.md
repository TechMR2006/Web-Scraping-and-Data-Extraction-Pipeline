# 🌐 Web Scraping and Data Extraction Pipeline

## 📌 Project Overview

This project demonstrates a complete **web scraping and data extraction pipeline** built using Python.  
The objective is to scrape product information from a publicly accessible website, clean the data, store it in a structured format, and generate a visual HTML summary dashboard.

The project fulfills all requirements of Task 4 (Intermediate Level).

---

## 🎯 Project Objectives

- Scrape product data from a website using BeautifulSoup
- Extract:
  - Product Name
  - Price
  - Rating
  - Description
  - Image URL
- Handle pagination
- Clean and standardize the data
- Save structured data into CSV
- Download product images locally
- Generate an HTML summary dashboard
